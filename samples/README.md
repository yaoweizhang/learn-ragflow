# samples/ — 共享样本文件

本教程**跨章复用**两个样本文件，用于保证每章的实验可复现、可对比。仓库里默认随附了一份**原创虚构**的演示样本，读者可以按自己需要替换成同规格的其他文档。

## 仓库随附的样本

### 1. `server_whitepaper.pdf`

- **格式**：PDF
- **语言**：中文
- **页数**：3–5 页
- **主题**：紫光恒越 R3630 G5 双路机架式服务器（虚构示例，非真实厂商白皮书）
- **必须包含**：至少 1 个表格

### 2. `disclosure.docx`

- **格式**：DOCX（Word 文档）
- **语言**：中文
- **内容**：财务披露文档（虚构示例，所有公司名、股票代码、人物、数字均为虚构）
- **必须包含**：多个段落 + 多个表格

## 替换样本

如果现有样本的字段抽取结果不理想，可以替换内容，但**保持文件名不变**（`server_whitepaper.pdf` 和 `disclosure.docx`，严格小写、下划线分隔）。文件名是硬编码的约定，每章都会读取这两个名字。

替换时建议保持的最低规格：

- PDF：3–5 页中文、含 1 个表格（用于 s11 表格抽取验证）
- DOCX：多个段落 + 多个表格（用于 s03 结构感知分块 + s10 GraphRAG 实体抽取）

## 验证

把文件放到 `samples/` 后，运行：

```bash
ls -lh samples/
# 应该看到：
# server_whitepaper.pdf
# disclosure.docx
```

随后进入任意章节目录运行 `code.py` 即可开始实验。

## 关于虚构内容的说明

仓库随附的两个样本是**为教程测试而原创的虚构文档**，不复制任何现实白皮书或真实财报：

- 公司名"青蓝科技"、股票代码"600XXX"、所有财务数据、人物、机构均为虚构
- 服务器型号"紫光恒越 R3630 G5"为教学示例，与任何现实产品无关
- 如与任何现实主体相似纯属巧合
